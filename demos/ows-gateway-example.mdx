The OWS gateway demo shows one application-layer use of Agon: metered Solana RPC access with OWS-signed payment updates.

## Overview

This example is useful if you want to see how the protocol can back repeated paid requests and x402-style API monetization.

It is not required for understanding the protocol itself. The protocol-only devnet demo remains the best reference for V4 settlement behavior and benchmarks.

## Related Concept

This is also one example of the broader [operator hub model](/core-concepts/operator-hubs): users pay one operator, and that operator coordinates paid access to downstream infrastructure on top of Agon.
