## Funding Your Account

### Primary Method — Direct Transfer

Send USDC on Solana to your unique deposit address. Agon detects the transfer via Helius webhooks and credits your balance automatically — typically within seconds.

```typescript
const account = await agon.getAccount()
console.log('Send USDC to:', account.depositAddress)
// No further action required — balance appears automatically
```

<Tip>
  A periodic sweep job also runs every 15 minutes as a failsafe. No deposit is ever silently lost even if a webhook is delayed.
</Tip>

### Fallback — Manual Confirmation

For compliance logging or when you need explicit confirmation of a specific deposit:

```http
POST /account/deposit
X-AGON-KEY: ag_xxxxx
Content-Type: application/json

{ "tx_signature": "5Ht8...xyz" }
```

This endpoint is fully idempotent — submitting the same signature twice returns `"already_credited": true`. No double-crediting is possible.

## Understanding Your Balances

Every account has two live balance fields:

| Field | Meaning |
|---|---|
| `available_balance` | Funds free to spend on API calls or withdraw on-chain |
| `reserved_balance` | Funds held for in-flight API calls |

**Total balance = `available_balance` + `reserved_balance`**

### How Balances Change

```
Deposit 100 USDC
  available: 100    reserved: 0

API call authorized ($1.00 reserved)
  available:  99    reserved: 1

Call succeeds → consume
  available:  99    reserved: 1   ← awaiting settlement

Settlement runs
  available:  99    reserved: 0   ← funds transferred to merchant

Call fails → release
  available: 100    reserved: 0   ← hold cancelled, funds returned
```

<Note>
  Reservations auto-expire after **5 minutes** if neither consumed nor released. Funds always return to `available_balance`. You are never charged for a failed request.
</Note>

## Currency

All API amounts are in USDC micro-units. **1 USDC = 1,000,000 micro-units.**

| USD Value | API Units |
|---|---|
| $0.001 | 1,000 |
| $0.01 | 10,000 |
| $0.10 | 100,000 |
| $1.00 | 1,000,000 |
| $10.00 | 10,000,000 |

## Withdrawals

Withdraw your `available_balance` to your owner wallet at any time:

```typescript
const result = await agon.withdraw(5) // 5 USDC
console.log('On-chain tx:', result.txSignature)
```

There is no lock-up, no minimum, and no delay. The USDC lands in your wallet in the same Solana transaction.
