Agon does not reject x402. It questions exact settlement on every interaction as the default model for high-frequency machine commerce.

## Overview

If every API call settles exactly, every call inherits the settlement cost of its own transaction.

That is fine for one-off, high-value payments. It becomes expensive when the same parties interact many times and each request is small.

## How Agon Differs

Agon treats exact per-payment x402 as a baseline to measure against.

The benchmark question is simple: how many exact-payment transactions would this scenario require if every call had to settle individually?

That is the baseline shown in Agon's benchmark artifacts.
