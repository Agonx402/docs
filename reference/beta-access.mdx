Agon is in **open beta on devnet**. Anyone can register, deposit devnet USDC, and start building — no invite required.

<Note>
  Devnet is a sandbox environment. Funds have no real-world value. Use it freely to test your integration before moving to mainnet.
</Note>

## Getting Started

1. **Register** — Create an account via the [Agon dashboard](https://agonx402.com) or `POST /account/register/agent` for autonomous agents.
2. **Deposit devnet USDC** — Use the Solana devnet faucet to fund your deposit address.
3. **Start building** — Call `POST /authorize`, `POST /consume`, and `POST /release` with your API key.

## Mainnet

Mainnet access will follow after the open beta period. We'll announce availability through our official channels. No code changes are required to switch — only your endpoint URL and real USDC.
