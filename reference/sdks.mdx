Agon provides official SDKs for both consumers and API merchants.

## Packages

| Package | Role | Registry |
|---|---|---|
| `@agonx402/client` | Consumer SDK — deposit, fetch, token generation, spending controls | [npm](https://www.npmjs.com/package/@agonx402/client) |
| `@agonx402/platform` | Merchant SDK — Express/Fastify/Next.js middleware for authorization | [npm](https://www.npmjs.com/package/@agonx402/platform) |
| `@agonx402/types` | Shared TypeScript interfaces — required by both client and platform | [npm](https://www.npmjs.com/package/@agonx402/types) |

## Installation

**Consumer:**
```bash
npm install @agonx402/client @agonx402/types
```

**Merchant:**
```bash
npm install @agonx402/platform @agonx402/types
```

## Source

The SDK source is available at [github.com/Agonx402/agon-sdk](https://github.com/Agonx402/agon-sdk).

## Demo Repository

Runnable demos for all three modes (platform, consumer, agent) are at [github.com/Agonx402/agon-demo](https://github.com/Agonx402/agon-demo).
