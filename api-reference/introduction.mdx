## Environments

| Environment | Base URL | Network | Use for |
|---|---|---|---|
| **Devnet** | `https://api-devnet.agonx402.com` | Solana Devnet | Development & testing |
| **Mainnet** | `https://api-mainnet.agonx402.com` | Solana Mainnet | Production |

<Note>
  Devnet uses test USDC — transactions are real API calls but no real money moves. Use the
  built-in [faucet](/recipes/testing-devnet) to get test USDC. Switch to the mainnet URL when
  you're ready to go live.
</Note>



## Authentication

| Header | Used by | Purpose |
|---|---|---|
| `X-AGON-KEY: ag_xxx` | Consumer | Account management, token creation |
| `X-AGON-TOKEN: eyJ...` | Consumer | Short-lived payment JWT sent to merchants |
| `Authorization: Bearer pk_xxx` | Merchant | Authorization endpoints |
